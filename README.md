# Визуализация диффузии металлов

Этот проект создан с использованием **Vue.js** и библиотеки **P5.js** для интерактивной визуализации процесса диффузии частиц различных металлов.  
Он демонстрирует, как температура влияет на кинетическую энергию и скорость движения частиц, а также показывает совместимость металлов при создании сплавов.

---

## Цель проекта

Главная цель — показать физический процесс диффузии наглядно, где каждая частица реагирует на изменение температуры.  
- **Кинетическая энергия и температура:** при повышении температуры частицы движутся быстрее, что ускоряет процесс диффузии.  
- **Скорость диффузии:** напрямую зависит от температуры и массы частиц.

---

## Особенности

- **Реалистичная анимация частиц:** движение частиц изменяется в зависимости от температуры.  
- **Интерактивные ползунки:** регулировка температуры в реальном времени.  
- **Совместимость металлов:** подсветка успешных и неэффективных комбинаций.  

---

## Эффективность сплавов

- **Эффективные сплавы:** обладают улучшенными механическими свойствами, высокой прочностью и долговечностью. Используются в строительстве, машиностроении, авиации, ювелирном деле.  
- **Неэффективные сочетания:** приводят к коррозии, хрупкости и разрушению структуры, что делает их непрактичными для использования.  

Примеры:  
- Золото + медь → долговечный ювелирный сплав.  
- Железо + углерод → сталь с высокой прочностью.  
- Золото + железо → нестабильный и хрупкий сплав.

---

## Технологии

- **Vue.js** — фронтенд-фреймворк для реактивного интерфейса.  
- **P5.js** — библиотека для анимации и рисования на Canvas.  
- **JavaScript / CSS / HTML** — логика и стили анимации.
